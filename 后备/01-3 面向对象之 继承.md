## <span id="2"> 2 什么是继承 </span>

1 Q: 什么是继承

A: S1 继承可以使子类 具有父类的属性和方法，而不用书写相同的代码
  
代码举例继承:

```js
arr = new Array()
arr.push(1)
arr.valueOf()   // 这里的valueOf方法就是 通过原型链继承自 Object.prototype对象上的
```

S2 JS中没有真正的类， 所以只能使用 构造函数产生的对象 来模拟类的实现

模拟类实现的代码:

```js
function Human(name){
  this.name = name
}

let human = new Human('jack')     // human 就是 Human类的 一个实例
```



## <span id="3"> 3 实现继承 </span>

1 Q: 如何用ES5方法 实现继承

A: 

S1 通过原型链实现继承，代码见下:

```js
function Human(name){
  this.name = name
}

Human.prototype.run = function(){         
  console.log(this.name + 'can run')
}

function Men(name){
  Human.call(this, name)   // 继承父类的实例属性,this用于指向创建的实例对象
  this.gender = 'male'
}

Men.prototype.playgame = function(){
  console.log('Men like playing games')
}

Men.prototype.__proto__ = Human.prototype  // 让子类的原型对象.__proto指向父类的原型对象


var man = new Men('klay')
```

S2 因为IE不支持`Men.prototype.__proto = Human.prototype`的写法，所以代码改为如下:

```js
function Human(name){
  this.name = name
}

Human.prototype.run = function(){         
  console.log(this.name + 'can run')
}

function Men(name){
  Human.call(this, name)        // 继承父类的实例属性,this用于指向创建的实例对象
  this.gender = 'male'
}

var f = function(){}
f.prototype = Human.prototype    // 让这个空构造函数的原型对象 完全等于Human的原型对象
Men.prototype = new f()         //  完全改变Men的原型对象，变成空构造函数的一个 实例对象

Men.prototype.playgame = function(){
  console.log('Men like playing games')
}

var man = new Men('klay')
```


2 Q: 如何用ES6方法 实现继承

A: S0 class关键字，本质是ES5方法的语法糖

S1 代码见下:

```js
class Human{         // 类名后直接跟大括号
  constructor(name){
    this.name = name
  }
  run(){
    console.log(this.name + 'can run')  
  }
}

class Men extends Human{    // extends关键字作用: 设置原型链
  constructor(name){
    super(name)             // super关键字作用: 调用继承 父类的实例属性/方法
    this.gender = 'male'
  }
  playgame(){
    console.log('Men like playing games')
  }
}
```
