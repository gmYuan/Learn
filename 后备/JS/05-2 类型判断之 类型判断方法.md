## <span id="2"> 2 如何判断变量是否是数组类型 </span>

1 Q: 如何判断变量是否是数组类型

A: 

S1 通过Array.isArray()方法

S2 通过instanceof

S3 通过Objdect.prototype.toString.call()方法

S4 通过constructor，如果xx是数组，那么 xx.constructor === Array

```js
function fn() {

  console.log(Array.isArray(arguments))   //false 因为arguments是类数组，不是数组
 
  console.log(Array.isArray([1,2,3,4]))   //true
    

  console.log(arguments instanceof Array) //fasle
    
  console.log([1,2,3,4] instanceof Array) //true
    

  console.log(Object.prototype.toString.call(arguments)) //[object Arguments]

  console.log(Object.prototype.toString.call([1,2,3,4])) //[object Array]
    

  console.log(arguments.constructor === Array) //false
 
  arguments.constructor = Array
  console.log(arguments.constructor === Array); //true
  
```



## <span id="3"> 3 如何进行 类型判断 </span>

1 Q: 如何判断一个变量的 类型

A:

S1 对于基本类型，可以使用 typeof 操作符（注意不能识别null）

S2 对于引用类型，可以使用 instanceof操作符

S3 对于所有JS常见类型，可以使用 Object.prototype.toString操作符


2 Q: 写一个函数，可以实现类型判断

A:

S1 代码逻辑步骤如下:

调用Object.prototype.toString判断传入的变量；

变成小写字符 + 根据空格分隔字符 + 获取index=1的字符串；

因为index=1的字符串会有一个]，所以需要去除掉最后一个字符( slice(0,-1)方法 )


S2 代码实现如下:

```js
function type(val) {
  return Object.prototype.toString.call(val).toLowerCase().split(' ')[1].slice(0,-1)

}

let str = "123"
type(str)             // "string"

*/

```
