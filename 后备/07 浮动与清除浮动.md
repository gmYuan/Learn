## <span id="2"> 2 浮动布局的特点 </span>

1 Q: 什么是浮动

A: S0 核心: 脱标 + 贴边

S1 被设置为浮动的元素，可以一直向左/向右移动，直到外边框碰到 包围块的内边界/另一个浮动元素的外边界
  
S2 因此，浮动元素之间 不会互相覆盖/重叠


2 Q: 浮动元素的特点

A: S0 核心: 字围 + 收缩 + 父坍塌

S1 特点1: 浮动元素会覆盖`在它之后渲染` 的非浮动元素，但非浮动元素内部的文字部分会围绕 浮动元素显示

S2 特点2: 浮动一个非替换元素(img)，如果不固定设置宽度，其宽度会自动收缩至内部 内容宽度

S3 特点3: 正常父元素的高度是自适应的，高度为其包含的内容总高度，而内部元素的浮动 会造成了父容器高度塌陷


3 Q: 浮动元素的影响

A: S0 核心: 兄弟元素 + 父元素 + 父元素同级元素

S1 对兄弟元素的影响:       覆盖了元素容器 + 兄弟元素文字内容会 围绕浮动元素

S2 对父元素的影响:         父容器高度坍塌，父容器高度只会包括 非浮动元素

S3 对父元素同级元素的影响: 父容器高度坍塌了，会影响同级元素的位置显示


## <span id="3"> 3 清除浮动的方法 </span>

1 Q: 如何清除浮动 

A: S0 ::afer伪类 + clear属性

S1 在浮动元素的父元素上，添加.clearfix类

S2 创建.clearfix::after伪类，设置如下代码

```css
.clearfix::after {
  display: block;
  content: '';
  clear: both
}
```
