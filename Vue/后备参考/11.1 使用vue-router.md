## <span id="2">2 vue-cli的安装 </span>

1 Q: 如何安装vue-cli

A: 

S1 全局安装vue­cli: npm install -g vue-cli

S2 初始化项目: vue init webpack my-project

S3 进入项目: cd my-project

S4 启动项目: npm run dev


## <span id="3">3 vue-router的引入和使用 </span>

1 Q: 如何安装vue-router

A: 

S1 安装vue-router: npm install --save vue-router

S2 路由的作用是: 不同的url，显示不同的页面/组件内容(有点类似于a链接)


2 Q: 如何创建基本路由


S1 安装路由 + 引入路由插件: import Router from 'vue-router'

S2 注册路由插件: `Vue.use(Router)`


S3 配置路由: const routes = [{ path:xxx, component:yyy },{}.....]

S4 创建路由实例: const router = new Router({routes,})

S5 挂载路由到vue根实例: new Vue( {......, router})


S6 确定视图加载的位置: <router-view></router-view>


3 Q: 如何创建 路由跳转

A: S1 在src的 router文件夹里创建index.js

S2 引入插件 + 注册插件 + 配置插件 + `导出插件实例` + 挂载到vue根组件中 + 引入index.js文件
