# 网络之 从URL到页面展示总览


Q1: 从URL到页面展示的过程是什么

A:

S1 DNS解析: 获取到 域名对应的IP地址

  - DNS解析流程是：依次读取  浏览器缓存 --> 操作系统缓存 --> hosts文件缓存 --> 路由缓存 --> 本地域名服务器 --> 根域名服务器 --> 顶级域名服务器 --> 权威域名服务器

S2 获取到 IP地址后，再通过3次握手 建立TCP连接

S3 请求方发送 HTTP请求

S4 服务器通过TCP确认收到请求 +  应用程序处理请求后，返回 HTTP报文

S5 浏览器通过TCP确认收到响应 + 浏览器开始解析 和 渲染页面

S6 请求都完成后，双方通过 TCP四次挥手  断开连接

S7 以上过程中，会有 CDN等中间缓存代理 和  各种负载均衡技术的 存在，从而减少 目标服务器的压力
